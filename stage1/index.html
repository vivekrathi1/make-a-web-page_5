Lesson 1:Intro To Web and HTML Notes:

Web Basics

Internet is the interconnection of networked computers, also known as the web. When we try to access a web page, we actually send an http request to this WEB(interconnection of Computers) through our Browser, which is connected to a collection of HTML documents on a server, which respond to our request and a page is rendered in our browser in the form ofcontent(videos,docs,images).

HTML Basics

HTML -Hyper text mark up language is nothing but the language, a language that contains the following:-

  • Text
  • Markup(the way the page is displayed)
  • References to Other Content::(Links to other webpages, Videos and images)

Tags/Attributes:

The language mainly consists of the following.

  • Tags:-define the layout of the content inside the page
  • Attributes:-refers to additional information regarding the contents

Block and Inline Elements:

Blocks represent a container element and Inline elements occupies only space within the tags, Also void tags, like image tags.

Lesson 2:Creating a structured document in HTML:

Basics of HTML and CSS

HTML and CSS are both languages.
Definitions
HTML controls the structure of the web page, like the structure of a house, which room is kitchen, bedroom,living room. etc.
CSS on the oher hand defines the style of a webpage, color of the walls and how things look in the house is something that would be done by CSS in a webpage.

HTML,What it consists of actually

HTML consists of syntax and rules and the basic word in the language is a tag. Browser knows how to interpret HTML tags into elements of a tree like structure using a convention known as DOM(Document Object Model).
Tags contain content, other tags or content and so on in a nested way. Each Tag defines the type of element inside of it.

CSS

CSS defines the style of the page, also has certain syntax and rules to change the way elements look on the page, the size of the font, color, background, borders, positions etc. In the exercise involving the developer tools we turned circles into squares by unchecking the border-radius attribute.

"Boxify" the Design

We also learnt about breaking down the webpage into pieces and how everything on a webpage is a box no matter what shape it might seem on the page.

Lesson 3: Applying Style to the page via CSS

Cascading Style Sheets

Each element on the HTML can have a class attribute and Style can be applied using two ways

  • Using The tag in the HTML.
  • Using the selector i.e. attributes to the tag, the class.

Inheritance:Concept of Re-usability

CSS Styles is inherited from parent classes to child classes.
Child classes retain their style and override anything in the parent classes.

Avoiding Repetition

In the CSS style sheet we can "group" elements having "common" style and give them style by grouping the common element,
Instead of repeating the same style for each element time and again.
Re-usability lets us re use the styles we have already defined and apply more styles as we put more content in the page.
It is advantageous, as we don't have to add style for each new element that we add. For e.g. I have defined a style for all concept-titles in my page in the CSS, If I add new concept-title as my page grows, I don't have to put a separate style for that, the style for class concept-title automatically gets applied to it.

Basics of Margins/Borders/Padding/Content

CSS uses margins/border/padding to keep the content distinct

  • Margins- Outer most container.
  • Border-container around the content.
  • Padding- Clears the area around the content.
  • Content- Image/Text.

Alignment Tricks

Border-box and flex box are the two approaches discussed in the lecture.
Flexible Box is an efficient way to align items and distribute the space among items in a container or div.
